# Tip Calculator

A responsive tip calculator built with React and TypeScript that allows users to input a bill amount, select or customize a tip percentage, and split the total across multiple people. It provides instant calculations for tip amount, total bill, and per-person share, with a clean, mobile-friendly interface and real-time validation for a smooth user experience.
